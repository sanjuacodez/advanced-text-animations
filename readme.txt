== Description ==
Advanced Text Animations for Elementor brings powerful, modular text animation widgets to your WordPress site. Animate your headlines, paragraphs, and more with both CSS and GSAP-powered effects, all with intuitive controls in the Elementor editor. Perfect for eye-catching hero sections, engaging callouts, and dynamic content.

== Features ==
* **CSS Animations:**
  - Wave, Rainbow, Glitch, Shake, Pulse, Fade, and more
  - Per-character and per-word animation modes
  - Customizable speed, repeat, yoyo, stagger, and delay
  - Style controls: typography, color, stroke, blend mode, hover color, etc.
  - Robust whitespace and space handling
* **GSAP Animations:**
  - Scroll-reveal (per-character/word, with repeat/yoyo options)
  - Typewriter effect with customizable blinking cursor
  - Advanced stagger, delay, and animation controls
  - Modular, per-widget GSAP engine (no global conflicts)
  - Works in both frontend and Elementor editor/preview

== Frequently Asked Questions ==
= Does this plugin require Elementor Pro? =
No, it works with the free version of Elementor.

= Can I use both CSS and GSAP animations together? =
You can choose either CSS or GSAP for each widget instance, but not both at the same time.

= Are the animations mobile-friendly? =
Yes, all animations are responsive and tested on mobile devices.
